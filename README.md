# Employee Management System

Java OOP project for managing employees inside departments.

## Features
- Add employees to departments
- Support multiple employee types:
  - Salaried Employee
  - Hourly Employee
  - Commission Employee
  - Base Plus Commission Employee
- Calculate employee earnings
- Display employee details

## Technologies Used
- Java
- OOP
- Maven
- NetBeans

## OOP Concepts Applied
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
